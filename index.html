  • Olimpiadas de Paris

    O projeto apresenta uma aplicação web desenvolvida para destacar os medalhistas brasileiros das Olimpíadas de Paris 2024. Ele prioriza tecnologias modernas para criar uma experiência interativa, incluindo buscas detalhadas, filtros por categoria (medalhas e esportes), e integração com links externos, tudo com foco em acessibilidade e responsividade.
